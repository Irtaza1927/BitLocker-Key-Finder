================================================================
  How to generate additional valid test keys
================================================================
Any 6-digit block B is a valid BitLocker block iff:
      B % 11 == 0                   AND
      B // 11 <= 65535  (0xFFFF)

Python one-liner:
    >>> import random
    >>> "-".join(f"{11*random.randint(0,65535):06d}" for _ in range(8))
